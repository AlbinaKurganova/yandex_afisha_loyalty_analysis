# Анализ лояльности пользователей Яндекс Афиши

## О проекте

Проект по исследовательскому анализу данных (EDA) о заказах пользователей Яндекс Афиши.  
**Цель** – выявить факторы, влияющие на возврат пользователей (совершение повторных заказов).

## Данные

- База данных PostgreSQL (data-analyst-afisha)
- Период: июль – октябрь 2024 года
- 290 611 заказов, 21 838 пользователей

## Ключевые результаты

| Показатель | Значение |
|------------|----------|
| Доля вернувшихся пользователей (2+ заказа) | **61.3%** |
| Доля лояльных пользователей (5+ заказов) | **28.3%** |
| Средняя выручка с заказа | 518.72 руб |
| Лучший день для первой покупки | Понедельник (63.0% возвратов) |

### Основные выводы

1. **Поведенческие характеристики** важнее демографических
2. **Критический период для удержания** – 7-14 дней после первой покупки
3. **Самые лояльные пользователи** – покупают 2-3 билета (73.6% возвратов)
4. **Спортивные мероприятия** – зона роста (низкий возврат 55.8%)
